# 트래포트 Homepage

## 기획
### 사이트 개요
1. One Page로 사이트를 구성한다.
2. 반응형으로 PC/모바일 구분없이 보여지도록 제작한다.
3. 적절한 애니메이션을 이용해서 사이트에 생동감을 준다.
4. 라이트/다크 모드를 지원하도록 제작한다.

### 사이트 구조
* assets = image, css, scss, js를 담는다.
* html = html을 담는다.
